<BR>An example of PD/PPD operation with CH32X035.
<BR>
<BR>x035_bas is an example of PD operation; developed on MountRiver.
<BR>When started, basic will run, so if you input PD xx, xxV will be output.
<BR>
<BR>x035_adp is an example of PPS operation; developed on Linux.
<BR>When started, basic will run, so if you input PPS xxxxx, xx.xxxV will be output.
<BR>
<BR>*After starting, you will need to unplug and plug the adapter once.
<BR>*In PPS mode, output will stop if there are no commands for a while.
<BR>
<BR>
<BR>CH32X035によるPD/PPD動作例。
<BR>
<BR>x035_basはPD動作例　MountRiver上で開発されている。
<BR>スタートするとbasicが動くので、PD xxと入力するとxxVが出力される。
<BR>
<BR>x035_adpはPPS動作例　Linux上で開発されている。
<BR>スタートするとbasicが動くので、PPS xxxxxと入力するとxx.xxxVが出力される。
<BR>
<BR>※スタート後にアダプターから、一度抜き差しが必要。
<BR>※PPSモードでは暫くコマンドが無いと、出力が停止する。
