<BR>An example of PD/PPS operation with CH32X035.
<BR>
<BR>CH32X035_bas is an example of PD operation; developed on MountRiver.
<BR>When started, basic will run, so if you input PD xx, xxV will be output.
<BR>Details of the program [ PPS For CH32X035 ](https://qiita.com/hi631/items/5613f73f4001111acf35)
<BR>
<BR>CH32X035_adp is an example of PPS operation; developed on Linux.
<BR>When started, basic will run, so if you input PPS xxxxx, xx.xxxV will be output.
<BR>Details of the program [ PPD For CH32X035 ](https://qiita.com/hi631/items/794336b0dedd5c15d64a)
<BR>
<BR>*After starting, you will need to unplug and plug the adapter once.
<BR>*In PPS mode, output will stop if there are no commands for a while.
<BR>
